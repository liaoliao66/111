## 四、计算公式汇总

### 4.1 翻牌与目标完成率

| 指标 | 公式 | 文本式 | 说明 |
|------|------|--------|------|
| 目标完成率 | \(\displaystyle \text{完成率}=\min\left(\frac{\text{实际值}}{\text{目标值}}\times100\%,\,999.9\%\right)\) | 完成率=min(实际/目标×100%, 999.9%) | 保留 1 位小数；`fmtPct` |
| 翻牌位数 | 按指标 pad 左补零 | 年吨 8、年 TEU 5、月吨 6、月 TEU 4、日吨 5、日 TEU 4 | `pad(n, length)` |

**演示目标 ※**（`TARGET_*`）：

| 常量 | 值 | 含义 |
|------|-----|------|
| `TARGET_BULK_ANNUAL` | 18,000,000 吨 | 年度散货+件杂货目标 |
| `TARGET_TEU_ANNUAL` | 90,000 TEU | 年度集装箱目标 |
| `TARGET_BULK_MONTH` | 1,500,000 吨 | 月度吨货目标 |
| `TARGET_TEU_MONTH` | 7,500 TEU | 月度 TEU 目标 |

### 4.2 船舶作业进度

| 指标 | 公式 | 文本式 |
|------|------|--------|
| 作业进度 | \(\displaystyle \text{进度}=\frac{\text{已完成量}}{\text{计划量}}\times100\%\) | 进度=已完成/计划×100%，取整展示 |

集装箱可按 TEU，散货/件杂货按万吨，须与计划量单位一致。

### 4.3 月度趋势展示规则

| 规则 | 说明 |
|------|------|
| 横轴 | 1月—12月 |
| 当月之后 | 值为 `null`，折线不连接（`trendThroughMonth`） |
| 双轴 | 左轴万吨：散货、件杂货；右轴 TEU：集装箱 |

### 4.4 集疏港占比

对单货类（集港量 \(G\)，疏港量 \(D\)）：

| 指标 | 公式 | 文本式 |
|------|------|--------|
| 集港占比 | \(\displaystyle P_g=\text{round}\left(\frac{G}{G+D}\times100\%\right)\) | 集港%=round(集港/(集港+疏港)×100%) |
| 疏港占比 | \(P_d=100-P_g\) | 疏港%=100-集港% |

环图扇区值为 **量**（万吨或 TEU），标签为 **占比**。

### 4.5 货种排行

| 规则 | 说明 |
|------|------|
| 排序 | 按作业量降序 |
| 展示 | 柱状图从左到右（或类目标签顺序）为 TOP→次位 |
| 单位 | 万吨 |

### 4.6 堆场利用率（占用当量）

| 步骤 | 公式 |
|------|------|
| 单货类占用当量 | \(eq_i = qty_i \times coeff_i\) |
| 已用当量合计 | \(EQ_{used}=\sum eq_i\) |
| 堆场利用率 | \(\displaystyle U=\min\left(\text{round}\left(\frac{EQ_{used}}{C_{cap}}\times100\%\right),\,100\%\right)\) | 
| 货类结构占比 | \(\displaystyle S_i=\text{round}\left(\frac{eq_i}{EQ_{used}}\times100\%\right)\) | 环图扇区比例 |

**演示参数 ※**：

| 货类 | 库存 qty | 单位 | 折算系数 coeff | 说明 |
|------|---------|------|---------------|------|
| 集装箱 | 4200 | TEU | 0.01 | 万标箱当量/TEU |
| 散货 | 29.17 | 万吨 | 1.2 | |
| 件杂货 | 12.78 | 万吨 | 1.797 | |
| 堆场容量当量 | `yardCapacityEq` = **128** | 万标箱当量 | — | 联调替换 |

环图中心大字：**堆场利用率 U%**；图例为各货类结构占比。

### 4.7 平均堆存天数

| 规则 | 说明 |
|------|------|
| 纵轴 | 堆存天数（天），演示 Y 轴 0—30 |
| 系列 | 集装箱、散货、件杂货各一条折线 |
| 月份 | 同 [4.3](#43-月度趋势展示规则)，仅展示至当前月 |
| 考核线 | 见 [3.5](#35-考核堆存天数)，`markLine` 虚线 |

**注意**：本指标为 **平均堆存天数**，不是堆场周转率；周转率公式见 [八、待确认](#八待业务确认口径) 若二期接入。

### 4.8 集装箱多式联运占比

| 指标 | 公式 | 文本式 |
|------|------|--------|
| 多式联运占比 | \(\displaystyle P_m=\text{round}\left(\frac{TEU_m}{TEU_m+TEU_o}\times100\%\right)\) | 占比=round(联运/(联运+其他)×100%) |

环图中心展示 \(P_m\)；扇区为「多式联运」「其他」TEU 量。

## 六、来源系统与刷新频率

### 6.1 TOS 模块映射

| TOS 业务模块 | 本屏指标 |
|-------------|---------|
| 生产统计 | 翻牌吞吐量、货类月度趋势、货种排行 |

### 6.2 刷新频率建议

| 数据类别 | 建议周期 |
|---------|---------|
| 时钟 | 1 s |
| 今日翻牌、在泊/待泊船 | 1—5 min |
| 本月/年翻牌 | 5—15 min |
| 月度趋势、排行、占比、堆场 | 1 h 或日批 |
| 目标值 | 日批或月初更新 |

### 6.3 演示与生产差异

- 图表与翻牌数据来自 `screen3.html` 内 **JavaScript 静态对象**，非实时 TOS。
- 联调时：接口返回 → 赋值演示变量 → 调用 `update()` / `applyTransportPeriod()` 等刷新函数。
- 单位统一：趋势图散货/件杂货为 **万吨**，翻牌吨货为 **吨**（注意 ×10000 换算）。

### 6.4 接口失败展示建议

| 场景 | 建议 |
|------|------|
| 翻牌接口失败 | 保留上次翻牌或显示 `—` |
| 单图表失败 | 该图区域「暂无数据」，不影响其他图 |
| 船舶表空 | 展示空表提示行 |
