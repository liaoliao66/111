# 武穴多式联运指挥运营中心 — 数据说明文档

> **对应页面**：`screen3.html`（集装箱运营专题屏）  
> **文档用途**：梳理本屏全部展示指标的数据名称、获取方式、来源模块及图表说明，供接口对接、验收与运维参考。  
> **来源系统简称**：**码头生产管理系统（TOS）** — 下文“来源位置”均指该系统内业务模块/菜单/接口字段。

## 数据总表

| 图表名称 | 数据名称 | 数据类型 | 数据来源 | 图表介绍 |
|---------|---------|---------|---------|---------|

| 顶部表头 | 星期 | 直接获取 | 前端本机系统时钟，按周索引映射为「星期×」 | 与日期、时间组合展示，增强时间感知。 |
| 顶部表头 | 当前时间 | 直接获取 | 前端本机系统时钟，格式：HH:mm:ss | 实时时钟，每秒刷新。 |
| 顶部表头 | 主标题「武穴多式联运指挥运营中心」 | 直接获取 | 大屏静态配置（页面写死，无需接口） | 平台/项目名称标识。 |
| 顶部表头 | 副标题（宏观展示·集装箱运营专题、设计能力说明） | 直接获取 | 大屏静态配置 / 项目基础参数库 | 说明本屏为集装箱专题视角，并展示设计年吞吐量 1800 万吨、集装箱作业区 TEU 能力 12 万标箱/年等背景信息。 |
| 吞吐量翻牌统计 | 统计年份 | 计算 | 规则：取系统当前公历年份，展示为「YYYY年」 | 与年累计指标联动，跨年时自动切换年份标签。 |
| 吞吐量翻牌统计 | 年累计吞吐量（吨） | 直接获取 | TOS → **生产统计** → 吞吐量汇总 → 年度累计（货类=散货+件杂货，计量单位：吨） | 展示当年度散货、件杂货合计吞吐量，翻牌 8 位数字；体现全港干散货/件杂货年度完成规模。 |
| 吞吐量翻牌统计 | 年累计集装箱吞吐量（TEU） | 直接获取 | TOS → **集装箱作业管理** → 箱量统计 → 年度累计 TEU | 展示当年度集装箱折算箱量（TEU），翻牌 6 位数字；专题屏核心指标之一。 |
| 吞吐量翻牌统计 | 本月累计吞吐量（吨） | 直接获取 | TOS → **生产统计** → 吞吐量汇总 → 当月累计（散货+件杂货，吨） | 展示自然月内散货、件杂货累计吞吐量，翻牌 4 位；用于观察当月生产节奏。 |
| 吞吐量翻牌统计 | 本月累计集装箱吞吐量（TEU） | 直接获取 | TOS → **集装箱作业管理** → 箱量统计 → 当月累计 TEU | 展示自然月内集装箱 TEU 累计，翻牌 4 位。 |
| 吞吐量翻牌统计 | 今日累计吞吐量（吨） | 直接获取 | TOS → **生产统计** → 吞吐量汇总 → 当日累计（散货+件杂货，吨） | 展示当日 0:00 至当前时刻散货、件杂货吞吐量，翻牌 4 位；支持准实时刷新。 |
| 吞吐量翻牌统计 | 今日累计集装箱吞吐量（TEU） | 直接获取 | TOS → **集装箱作业管理** → 箱量统计 → 当日累计 TEU | 展示当日集装箱 TEU 累计，翻牌 4 位；与吨货指标同屏对照。 |
| 在港船舶 | 船名 | 直接获取 | TOS → **船舶调度** → 在港船舶动态 → 船名 | 在港船舶列表主键展示字段，支持多条船舶轮播/滚动。 |
| 在港船舶 | 航线 | 直接获取 | TOS → **船舶调度** → 航次/船舶动态 → 始发港、目的港（展示规则：始发港→目的港） | 描述船舶当前航次流向，如「上海→武穴」。 |
| 在港船舶 | 箱量 | 直接获取 | TOS → **集装箱作业管理** → 船舶作业计划/实际 → 计划或实际箱量（TEU/自然箱，按接口约定） | 本船本次作业涉及集装箱数量，专题屏以箱量替代宏观屏「运单数」。 |
| 在港船舶 | 装卸 | 直接获取 | TOS → **船舶作业管理** → 作业指令 → 作业类型（装/卸） | 标识当前船舶正在装船还是卸船。 |
| 在港船舶 | 作业进度（%） | 计算 | 规则：作业进度(%) = 已完成箱量 ÷ 计划箱量 × 100%，结果取整；数据来源：TOS → 船舶作业管理 → 作业进度 | 以进度条+百分比展示单船作业完成度；100% 表示作业完成。 |
| 货类吞吐量趋势（月度） | 月份（1月—12月） | 直接获取 | TOS → **生产统计** → 按月维度字典（自然月） | 折线图横轴，展示全年 12 个月趋势。 |
| 货类吞吐量趋势（月度） | 散货月度吞吐量 | 直接获取 | TOS → **生产统计** → 分货类吞吐量 → 散货 → 按月汇总（万吨） | 左 Y 轴（万吨）折线之一，反映散货月度波动。 |
| 货类吞吐量趋势（月度） | 件杂货月度吞吐量 | 直接获取 | TOS → **生产统计** → 分货类吞吐量 → 件杂货 → 按月汇总（万吨） | 左 Y 轴（万吨）折线之一。 |
| 货类吞吐量趋势（月度） | 集装箱月度吞吐量 | 直接获取 | TOS → **集装箱作业管理** → 按月箱量汇总（TEU） | 右 Y 轴（TEU）折线；与吨货分轴展示，避免量纲混用。 |
| 集疏港量占比 | 散货集港量 | 直接获取 | TOS → **集疏港管理** → 集疏港作业统计 → 货类=散货 → 集港量（万吨） | 散货环图「集港」扇区数值。 |
| 集疏港量占比 | 散货疏港量 | 直接获取 | TOS → **集疏港管理** → 集疏港作业统计 → 货类=散货 → 疏港量（万吨） | 散货环图「疏港」扇区数值。 |
| 集疏港量占比 | 散货集港占比（%） | 计算 | 规则：集港占比 = 集港量 ÷ (集港量 + 疏港量) × 100%，四舍五入取整；环图中心展示该值 | 散货环图中心百分比及「集港」标注。 |
| 集疏港量占比 | 件杂货集港量 | 直接获取 | TOS → **集疏港管理** → 货类=件杂货 → 集港量（万吨） | 件杂货环图集港扇区。 |
| 集疏港量占比 | 件杂货疏港量 | 直接获取 | TOS → **集疏港管理** → 货类=件杂货 → 疏港量（万吨） | 件杂货环图疏港扇区。 |
| 集疏港量占比 | 件杂货集港占比（%） | 计算 | 规则：集港量 ÷ (集港量 + 疏港量) × 100% | 件杂货环图中心占比。 |
| 集疏港量占比 | 集装箱集港量 | 直接获取 | TOS → **集疏港管理** / **集装箱作业管理** → 货类=集装箱 → 集港箱量（TEU） | 集装箱环图集港扇区。 |
| 集疏港量占比 | 集装箱疏港量 | 直接获取 | TOS → **集疏港管理** / **集装箱作业管理** → 疏港箱量（TEU） | 集装箱环图疏港扇区。 |
| 集疏港量占比 | 集装箱集港占比（%） | 计算 | 规则：集港箱量 ÷ (集港箱量 + 疏港箱量) × 100% | 集装箱环图中心占比。 |
| 堆存量占比 | 集装箱堆场库存量 | 直接获取 | TOS → **堆场管理** → 库存查询 → 货类/堆场类型=集装箱 → 当前库存（吨或标箱折算，按库存计量口径） | 环图扇区「集装箱」；与散货、件杂货共同构成堆存结构。 |
| 堆存量占比 | 散货堆场库存量 | 直接获取 | TOS → **堆场管理** → 库存查询 → 货类=散货 → 当前库存 | 环图扇区「散货」。 |
| 堆存量占比 | 件杂货堆场库存量 | 直接获取 | TOS → **堆场管理** → 库存查询 → 货类=件杂货 → 当前库存 | 环图扇区「件杂货」。 |
| 堆存量占比 | 集装箱堆存占比（%） | 计算 | 规则：集装箱库存 ÷ (集装箱+散货+件杂货库存) × 100%；环图中心展示集装箱占比 | 环形图中心大号百分比为集装箱占三类堆存总量比例。 |
| 堆存量占比 | 各货类堆存占比（%） | 计算 | 规则：单货类库存 ÷ 三类库存合计 × 100%（ECharts 饼图自动计算 `{d}%`） | 图例及 tooltip 展示各货类占比。 |
| 货种排行榜 | 货种名称 | 直接获取 | TOS → **生产统计** → 分货种作业量排行 → 货种名称 | 柱状图横轴类目（如原煤、木材、件杂、钢材等）。 |
| 货种排行榜 | 货种作业量 | 直接获取 | TOS → **生产统计** → 分货种作业量 → 作业吨数（万吨） | 柱状图柱高及柱顶标签；接口返回后按降序排列取 TOP。 |
| 货种排行榜 | 排行顺序 | 计算 | 规则：按货种作业量从高到低排序后展示 | 保证第一名在最左侧（或最上，依图表配置）。 |
| 堆场周转率（月度） | 月份（1月—12月） | 直接获取 | TOS → **堆场管理** → 周转率统计 → 按月维度 | 折线图横轴。 |
| 堆场周转率（月度） | 集装箱月度周转率 | 计算 | 规则：周转率(%) = 期间出库量 ÷ 期间平均库存 × 100%（或按 TOS 堆场模块既定公式）；货类=集装箱，按月汇总 | 反映集装箱堆场库存周转效率，Y 轴为百分比。 |
| 堆场周转率（月度） | 散货月度周转率 | 计算 | 规则：同上，货类=散货，按月汇总 | 散货堆场周转趋势折线。 |
| 堆场周转率（月度） | 件杂货月度周转率 | 计算 | 规则：同上，货类=件杂货，按月汇总 | 件杂货堆场周转趋势折线。 |
| 公铁水作业量 | 月份（1月—12月） | 直接获取 | TOS → **多式联运管理** / **集疏运管理** → 按月维度 | 月度折线图横轴。 |
| 公铁水作业量 | 公路月度作业量（TEU） | 直接获取 | TOS → **多式联运管理** → 公铁水联运统计 → 运输方式=公路 → 按月箱量/作业量（TEU） | 公路联运折线；本屏专题以 TEU 为统一量纲。 |
| 公铁水作业量 | 铁路月度作业量（TEU） | 直接获取 | TOS → **多式联运管理** → 运输方式=铁路 → 按月 TEU | 铁路进港专用线相关箱量统计。 |
| 公铁水作业量 | 水路月度作业量（TEU） | 直接获取 | TOS → **多式联运管理** → 运输方式=水路 → 按月 TEU | 水路转运箱量，体现铁水联运链条。 |
| 多式联运占比 | 多式联运订单箱量 | 直接获取 | TOS → **多式联运管理** → 订单统计 → 多式联运业务标记=是 → 箱量合计（TEU） | 饼图扇区「多式联运」。 |
| 多式联运占比 | 其他订单箱量 | 直接获取 | TOS → **多式联运管理** → 订单统计 → 非多式联运单 → 箱量合计（TEU） | 饼图扇区「其他」。 |

## 补充说明

### 1. 当前演示页与生产接口

- 现版 `screen3.html` 中部分指标为**前端模拟数据**（如翻牌定时自增、图表静态数组），联调时需按上表替换为 TOS 接口返回值。
- 翻牌区「本月/今日」吨货展示若与接口单位不一致，应在接口层或展示层统一换算规则（如：吨保留整数、万吨保留一位小数），避免与趋势图量纲不一致。

### 2. TOS 模块映射汇总

| TOS 业务模块 | 本屏涉及指标 |
|-------------|-------------|
| 生产统计 | 年/月/日吞吐量，分货类月度趋势，货种排行榜 |
| 船舶调度 / 船舶作业管理 | 在港船舶列表、装卸、作业进度 |
| 集装箱作业管理 | TEU 累计、在港箱量、集装箱集疏港、月度箱量趋势 |
| 堆场管理 | 堆存量占比、堆场周转率 |
| 集疏港管理 | 集疏港量、集港/疏港占比 |
| 多式联运管理 | 公铁水作业量、多式联运订单占比 |

### 3. 刷新频率建议

| 数据类别 | 建议刷新周期 |
|---------|-------------|
| 时钟类 | 1 秒 |
| 今日累计、在港船舶 | 1—5 分钟 |
| 本月/年累计 | 5—15 分钟 |
| 月度趋势、排行榜、占比类 | 1 小时或日批更新 |
